## K線
`公式 K線 = N天的收盤價總和/N天`

![K線]( https://www.cmoney.tw/learn/cmstatic/learn/44/20200415151444639.png )
> 
- 短期均線： 5、10日
- 中期均線： 20、60日
- 長期均線： 120、240日
### 1. 操作
- 收盤價>5日均線>10日均線 :漲( 5日由下往上穿過10日)
- 收盤價<5日均線<10日均線 :跌( 5日由上往下穿過10日)
- 5日均線<收盤價<10日均線 :平穩
- 當股價跌破5日線，要減碼賣出，一旦跌破10日線，要全部賣出，暫作觀望。
### 2. 作用
- K線隨時間連在一起，就可以看出股價過去的走勢圖
